LCD-paneelin funktioiden käyttöohjeet

Tässä esitettävä funktiopaketti on koodattu helpottamaan Arduino käyttöä ohjelmoinnin harjoitustehtävissä. Näyttönä oletetaan olevan LCD Keypad Shield.

Sisältö:

  1. Painikkeet
  2. AD muunnin
  3. Ledin käyttö
  4. LCD-paneeli

Painikkeet

Näyttökortilla on viisi painiketta, joiden luenta tapahtuu funktiolla unsigned char GetKey();

Se palauttaa jonkin seuraavista arvoista

#define NO_KEY 99

#define IDK_RIGHT 1

#define IDK_UP 2

#define IDK_DOWN 3

#define IDK_LEFT 4

#define IDK_SELECT 5

AD - muunnin

Kortin analogiatuloihin voi kytkeä jännitteen välille 0..5V. Sen arvo näkyy numeerisena välillä 0..1023. Mitattava tulo valitaan funktiolla InitADC. Muuttujalle portti voidaan antaa arvo 1..5, koska kanava 0 on varattu painikkeiden lukua varten.

void InitADC(char portti);
int ReadADC(void);

Esimerkki:

int nRawValue; int temperature;

InitADC(1);

nRawValue = ReadADC();

temperature = (nRawValue  500)*10/85;

 

Ledin käyttö

Kortilla on yksi ledi, jota voi sytyttää ja sammuttaa. Funktiot:

#define LED_MOTHER_BOARD 0

void LedOn( char number );

void LedOff(char number);

LCD-paneeli

LCD-paneelissa on kaksi 16 merkin riviä. Sisäisessä muistissa riville mahtuu 40 merkkiä, mikä mahdollistaa näytön vierittämisen molempiin suuntiin.

void lcd_init

( 

uint8_t 

dispAttr

 ) 

 

 

Alustaa näytön ja asettaa kursorin päälle tai pois.

Tätä on käytettävä ennen muita funktioita.

Parametrit:

dispAttr 

LCD_DISP_OFF näyttö päältä
LCD_DISP_ON näyttö päällä, kursori päältä
LCD_DISP_ON_CURSOR näyttö päällä, kursori päällä
LCD_DISP_ON_CURSOR_BLINK näyttö päällä, kursori päällä, vilkkuu

Palauttaa:

ei mitään

(

 )

Parametrit:
